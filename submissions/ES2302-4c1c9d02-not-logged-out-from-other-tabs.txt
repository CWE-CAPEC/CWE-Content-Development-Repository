SUBMISSION TYPE: Software

NAME: Not logged out from other tabs during logging out in current tab

DESCRIPTION:

During logging out action user is not logged out from other opened in the
same browser's window tabs from the same app or webapp. User can thinking
that all data are safe, and don't have to or don't want to check other
tabs, but bad actor (other user of device - public library, org/company,
and so on, still can see sensitive data if check other tabs after logged
out action). That is especially dangerous issue for services like invoices,
payments and other situations where are containing sensitive data. This
weakness is dangerous at all, because even initially not sensitive data, can
be next useful for bad actor - depends on situation.
